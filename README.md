An interactive dashboard built using Microsoft Power BI, Jupyter Notebook, and Excel to analyze housing prices through KPIs, charts, and interactive filters.

## Features
- Predict house prices using machine learning models in Python.
- Visualize data with Excel dashboards.
- Present insights with Power BI interactive charts.
- Includes dataset, notebook, and presentation files.

## Files
- `House_Price_Prediction.ipynb` → Jupyter Notebook with ML workflow
- `HousePricePrediction.xlsx` → Dataset
- `House-Price-Prediction-Dashboard.xlsx` → Excel dashboard
- `House-Price-Prediction-Dashboard-using-PowerBI.pptx` → Power BI presentation
